Hackathon Clima View

Desafie-se e explore o mundo do clima!

Participe do Hackathon Clima View, um evento exclusivo para os alunos do IFTM - Campus Ituiutaba. Desenvolva uma plataforma interativa que explore dados climáticos em tempo real e exiba suas habilidades em frontend e integração de APIs.

Participe Agora

Descrição do Desafio

O Hackathon "Clima View" é exclusivo para alunos do IFTM campus Ituiutaba. Seu objetivo é criar uma plataforma que permita a exploração de dados climáticos em tempo real. Utilize APIs de mapas e clima para criar uma experiência dinâmica e envolvente.

Objetivo do Desafio

Criar uma aplicação frontend que mostre informações climáticas em um mapa interativo, permitindo que os usuários vejam o clima em qualquer cidade ou região. A interface deve ser intuitiva, responsiva e visualmente atraente, oferecendo dados detalhados sobre o clima.

Requisitos do Desafio

  • Interface de Usuário: Desenvolver uma interface moderna e responsiva com temática de clima e natureza.
  • Funcionalidades: Exibir um mapa interativo, com informações climáticas ao selecionar uma localização específica.
  • APIs: Utilizar uma API de mapas e uma API de dados climáticos para exibir informações em tempo real.
  • Bônus: Transições suaves e campo de busca para uma experiência imersiva.

Entrega do Produto

Envie o código completo em um repositório público no GitHub, com README detalhado incluindo nome completo, curso, período e instruções de como rodar a aplicação.
